---
title: Prerequisites
description: Tools you need installed before running Zephyr locally.
---

## Required

- Git
- Bun (package manager/runtime)
- Node.js LTS (some tooling may rely on Node APIs)
- Docker Desktop (for the Docker workflow)

## Optional but useful

- VS Code with TypeScript and MDX extensions
- Prisma CLI
- Redis and Postgres clients

Verify versions:

```bash
bun --version
docker --version
```
